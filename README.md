# Mini Blockchain

Proyecto académico enfocado en la simulación de una blockchain utilizando hashes SHA-256 y almacenamiento en MySQL para validar la integridad de la información.

## Tecnologías utilizadas
- C#
- MySQL
- SHA-256
- Visual Studio

## Funcionalidades
- Creación de bloques enlazados
- Validación de integridad criptográfica
- Detección de alteración de bloques
- Persistencia de datos en MySQL

## Objetivo
Comprender el funcionamiento básico de blockchain y la aplicación de funciones hash para garantizar la integridad de la información.
